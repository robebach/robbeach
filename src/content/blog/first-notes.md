This is a sample blog post written in Markdown.

The blog index now reads posts from `src/content/blog`, so adding a post is just a matter of creating another `.md` file with frontmatter.

## What belongs here

- Short notes
- Project updates
- Longer essays
- Technical write-ups

Keep the file name readable. It becomes the post URL.
